# results/

本文件夹存放**实验结果**，包括数值记录、图表、日志等。

> ⚠️ 注意：大型模型权重文件请勿上传 GitHub，建议上传至 Hugging Face 或其他平台后在主 README 中提供下载链接。

## 文件夹结构

```
results/
├── tables/        ← 实验数值结果（.csv 或 .txt）
├── figures/       ← 图表（.pdf 或 .png）
└── logs/          ← 训练/评测日志（不上传 GitHub）
```
